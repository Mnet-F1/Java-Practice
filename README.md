## 학습 방법
1. 문제를 풀어본다 (해결 과정 보면서)
2. 해당 개념을 잘 모르겠다면 개념부터 다시 공부한다
3. 문제 해결이 안된다면 힌트를 본다
4. 2시간 이상 고민 했는데 해결 못하겠다면 답을 먼저 쭉 훑어본다
5. 다시 풀어본다
6. 그래도 안되겠으면 따라서 쳐본다
7. 안보고 한번 더 풀어본다
